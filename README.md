# 목포 티맵 보이스 Overlay

티맵은 묵음으로 두고, 접근성 서비스가 티맵 길안내 문구만 감지해서 목포/전라도 느낌의 맛깔난 TTS 문장으로 다시 읽어주는 테스트 앱입니다.

## V1.1 Mokpo 변경점
- 티맵 화면 전체 읽기 방지 필터 유지
- 좌회전/우회전/유턴/직진/단속/도착 안내를 목포 사투리 스타일로 변환
- 앱 아이콘 추가
- 앱 이름: 목포 티맵 보이스

## 사용법
1. APK 설치
2. 앱 실행
3. 접근성 설정에서 `목포 티맵 보이스` 켜기
4. 티맵 길안내 음성은 묵음 처리
5. 티맵 실행 후 테스트

## GitHub Actions
저장소에 push하면 `.github/workflows/android-build.yml`로 APK가 자동 빌드됩니다.
