# SM3 Audio Suite V2.0 Practical

실사용 안정화 방향의 Android 프로젝트입니다.

## V2.0 변경점
- 티맵 접근성 감지 강화
  - 이벤트 텍스트 + 소스 노드 + 루트 노드 동시 확인
  - 거리와 방향이 분리되어 잡힐 때 합쳐서 안내
  - 화면 전체 읽기 방지 필터 강화
  - 중복 안내 9초 차단
- 폴더 선택 시 음악 자동 수집/정렬/첫 곡 자동 재생
- 스펙트럼 아날라이저 포함
- 32밴드 EQ UI 포함
- 리버브 조절 포함
  - 기본 추천: 공간감 38 / 룸 28 / 잔향 34
- 목포 아재 / 목포 아주머니 / 차분 / 표준어 음성 선택
- GitHub Actions APK 빌드 포함

## 사용 순서
1. GitHub에 압축 해제한 내용을 업로드
2. Actions에서 APK 빌드
3. 기존 앱 삭제 후 APK 설치
4. 앱 실행 후 접근성 설정에서 `SM3 Audio Suite` 활성화
5. 티맵 자체 안내 음성은 묵음 처리

## 주의
티맵 Android Auto 화면에서는 접근성으로 안내 문구가 안 잡힐 수 있습니다. 이 경우 휴대폰/헤드유닛의 티맵 화면 기준으로 테스트하세요.
