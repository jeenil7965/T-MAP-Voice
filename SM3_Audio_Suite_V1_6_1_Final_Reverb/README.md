# SM3 Audio Suite V1.6.1 Final Audio + Reverb

실행용 Android 프로젝트입니다.

## 포함 기능
- 티맵 안내 접근성 감지 및 목포/표준 TTS 안내
- 폴더 선택 시 음악 파일 자동 검색 및 자동 재생
- 이전/다음/재생/정지/셔플
- 스펙트럼 아날라이저
- 32밴드 EQ UI
- 리버브 조절: 공간감, 룸 크기, 잔향 길이
- 기본 리버브 추천값: 공간감 38%, 룸 28%, 잔향 34%
- GitHub Actions APK 빌드

## 사용
1. GitHub에 압축 해제한 내용을 업로드
2. Actions에서 APK 빌드
3. 앱 설치 후 접근성에서 SM3 Audio Suite 활성화
4. 티맵 자체 음성은 묵음 처리
